# IFRMS
A working complete web application to integrate fitness management and resource management for gyms.

# INTRODUCTION
This is done as a team project for DBMS course.The team consists of 4 members including me and my three friends.
The aim of the project is to create a complete working web application dedicated to manage both resources of gyms and customers and staff of the gym.
This project we are doing is inspired by the problem faced by some of my gym going friends one who is in my team for this project.The problem they face is that they are required to physically go to the college gym to renew their membership every month, and the gym note the money transactions in a note book.
So we decided to create a database and dedicatefd web application to solve this problem though the project we are doing is not a complete solution to the problem we addresed because we are developing this web application for resource and membership management only.
we are thinking to integrate online payment for membership renewal from the application itself feature in future versions.
